# grunt-hogan-example

shows how to use grunt-hogan

## Getting Started
Install the module with: `npm install grunt-hogan-example`

```javascript
var grunt_hogan_example = require('grunt-hogan-example');
grunt_hogan_example.awesome(); // "awesome"
```
